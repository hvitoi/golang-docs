# Go CLI

```sh
# Compile go code
go build "main.go" # "main" executable generated can be run

# Compile & execute go code
go run "main.go"
go run "main.go" "deck.go" # run multiple files

# Format all the code contained in the current directory
go fmt

# Compile and install a package
go install

# Download raw source code of someone else's package
go get

# Run tests associated with the current project
go test
```
